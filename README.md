# CreamCo

CreamCo is a self-learning UI/UX project designed in Figma, showcasing an interactive ice cream ordering experience.

The project focuses on user flow, micro-interactions, and animated transitions, allowing users to browse flavors, view details, select scoops, and add items to the cart.

**Tools:** Figma  
**Role:** UI/UX Designer (Self-learning)
